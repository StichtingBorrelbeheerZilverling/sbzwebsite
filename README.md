# sbzwebsite
Website of Stichting Borrelbeheer Zilverling

## Development

### New Features and Updates

When you added a new feature or updated something else to the website, and you want to put it online. 
Make sure to merge the changes of your development branch into the `master` branch first. After that merge `master` into the `production` branch.
Since `production` is the current branch for deployment.
You can also skip the merge into the `master` branch. But this way the `production` & `master` branch will always be up-to-date with the latest changes and features. 
And won't have big differences for if deployment using the `production` branch gets moved to deployment using the `master` branch.
